Divisibility

If ddd﻿ is a divisor of nnn﻿, we can express nnn﻿ as n=k⋅dn = k \cdot dn=k⋅d﻿.

But, what if ddd﻿ is not a divisor of nnn﻿?

For convenience, let’s assume both are positive.

If ddd﻿ does not divide nnn﻿, we must not be able to reach 000﻿ by repeated subtractions. So, there exist some consecutive steps as follows:

Let r=n−(s−1)dr = n - (s-1)dr=n−(s−1)d﻿. We can combine the inequalities r>0r > 0r>0﻿ and r−d<0r - d < 0r−d<0﻿ as follows:

Why do you think rrr﻿ is named so?

It stands for the term rrr﻿emainder!

A remainder can be thought of as the remained part left of nnn﻿ in the last step of reaching 000﻿. It is “remained” because we cannot subtract ddd﻿ from it anymore.

We can use the remainder to write an equation similar to n=kdn = kdn=kd﻿:

Did you realize that when r=0r = 0r=0﻿, it looks similar to n=kdn = kdn=kd﻿?

Voila!

ddd﻿ divides nnn﻿ if and only if r=0r = 0r=0﻿.

It isn’t over yet. We don’t need the naive subtraction process to find rrr﻿. Our computers have a powerful operation for that: the modulo operation! It uses the operator % to find rrr﻿, i.e., n % d == r.

The result is called modulus.

Below are some example modulo operations in C++:

16  % 1 == 0
17  % 5 == 2
6   % 0 == error!
4   % 4 == 0
-17 % 5 == -2 (results in 3 in Python!)
4   % 145213421421 == 4

Using this operation, we can find the divisors of nnn﻿ by iterating over 1−n1 - n1−n﻿ and checking whether the remainder is 000﻿:

// https://ideone.com/k2WMUG
#include <iostream>
#include <vector>
using namespace std;
 
int main() {
	int n = 123456;
	vector<int> positive_divisors;
	for(int i = 1; i <= n; i++){
		if(n % i) continue; // r > 0
		// i is a divisor, do whatever you want!
		positive_divisors.push_back(i);
	}
	int n_positive_divisors = positive_divisors.size();
	cout << "n: " << n << "\n";
	cout << "Number of positive divisors: " << n_positive_divisors << "\n";
	cout << "Number of all divisors: " << 2 * n_positive_divisors << "\n";
	cout << "Positive divisors:\n";
	for(int d: positive_divisors){
		cout << d << " ";
	}
	cout << "\n";
}
# https://ideone.com/0Cj8MK
n = 123456
positive_divisors = []
for i in range(1, n + 1):
    if n % i:  # r > 0
        continue
    # i is a divisor!
    positive_divisors.append(i)

n_positive_divisors = len(positive_divisors)
print(f"n: {n}")
print(f"Number of positive divisors: {n_positive_divisors}")
print(f"Number of all divisors: {2*n_positive_divisors}")
print("Positive divisors:")
print(" ".join(map(str, positive_divisors)))
Primes

A prime number is a positive integer that has exactly 222﻿ different positive divisors (or 444﻿ divisors in total).

It is an integer’s purest form because a prime is not a “combination” (product) of smaller integers other than 111﻿.

Non-prime integers also have their own name: composite! Because they are composed of smaller integers greater than 111﻿.
