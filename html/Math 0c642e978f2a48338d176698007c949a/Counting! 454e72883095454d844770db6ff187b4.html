Factorials

Factorials are the building blocks of counting. They are the most atomic elements after integers when it comes to counting.

The term ‚Äúfactorial‚Äù needs a non-negative integer nnnÔªø to be defined, that is, ‚Äúfactorial of nnnÔªø‚Äù is defined as the product of all positive integers up to nnnÔªø. It is denoted with n!n!n!Ôªø:

Additionally, 0!=10! = 10!=1Ôªø. They can also be defined recursively:

They are used to describe more complex stuff, such as binomial coefficients.

n!n!n!Ôªø itself represents the number of ways to arrange nnnÔªø different items, like the integers from 111Ôªø to nnnÔªø (permutations!).

#include <bits/stdc++.h>

using ll = long long;

ll factorial(int n) {
    ll res = 1;
    for (int i = 1; i <= n; i++) {
        res *= i;
    }
    return res;
}

ll factorial_recursive(int n) {
    if (!n)
        return 1;
    return n * factorial_recursive(n - 1);
}

        cout << " ";
        cout << factorial(i) << " (iterative), ";

    }
}

Binomial Coefficients, a.k.a. (nk)\binom{n}{k}(kn‚Äã)Ôªø

The most memorable signature from the high school has been (nk)\binom{n}{k}(kn‚Äã)Ôªø (nnnÔªø choose kkkÔªø) which basically stands for the number of ways to choose kkkÔªø different items among nnnÔªø different items. Literally, every answer to a counting problem can be expressed using binomial coefficients in a reasonable manner.

As mentioned previously, a binomial coefficient (nk)\binom{n}{k}(kn‚Äã)Ôªø (0‚â§k‚â§n)(0 \le k \le n)(0‚â§k‚â§n)Ôªø can be defined involving factorials:

And, it is undefined (or 000Ôªø) for k>nk > nk>nÔªø. We consider it 000Ôªø in our computations.

It can also be defined recursively:

One way to derive this formula is to derive the problem of finding the number of ways to choose kkkÔªø different items among nnnÔªø different items into 222Ôªø separate cases depending on the last item:

  1. We choose the last item, then choose k‚àí1k-1k‚àí1Ôªø items among the first n‚àí1n-1n‚àí1Ôªø items ‚Üí(n‚àí1k‚àí1)\to \binom{n-1}{k-1}‚Üí(k‚àí1n‚àí1‚Äã)Ôªø,
  1. We don‚Äôt choose the last item, choose kkkÔªø items among the first n‚àí1n-1n‚àí1Ôªø items ‚Üí(n‚àí1k)\to \binom{n-1}{k}‚Üí(kn‚àí1‚Äã)Ôªø.

Hence the formula. There are several other ways, some of which involve Pascal‚Äôs triangle.

Implementation
// https://ideone.com/tlbrtz
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

ll factorial(int n) {
    ll res = 1;
    for (int i = 1; i <= n; i++) {
        res *= i;
    }
    return res;
}

// We abbreviate "binomial coefficient" as "C" standing for "Combination".
ll C(int n, int k) {
    if (k > n)
        return 0;
    // numerator == n!
    // denominator = (n-k)! * k!
    ll numerator = factorial(n), denominator = factorial(n - k) * factorial(k);
    return numerator / denominator;
}

void print_binomial_coefficients(int n) {
    ll C_table[n + 1][n + 1];
    for (int i = 0; i <= n; i++) {
        for (int j = 0; j <= n; j++) {
            if (i < j) {
                C_table[i][j] = 0;
                continue;
            }
            if (!j) {
                C_table[i][j] = 1;
                continue;
            }
            C_table[i][j] = C_table[i - 1][j - 1] + C_table[i - 1][j]; // even if i-1 < j, C_table[i-1][j] = 0, no problem here!
        }
    }
    cout << "Binomial coefficients calculated w/ factorial (left) vs. w/ recursive formula (right)\n";
    int n_examples = 10;
    for (int i = 0; i < n_examples; i++) {
        cout << i + 1 << ".\t";
        int rand_n = rand() % n;
        int rand_k = rand() % (rand_n + 1);
        cout << "n = " << rand_n << ", k = " << rand_k << ", C(n, k): " << C(rand_n, rand_k) << " vs. " << C_table[rand_n][rand_k] << "\n";
    }
}

int main() {
    int n = 30;
    print_binomial_coefficients(n);
}
